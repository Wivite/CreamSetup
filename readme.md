## How to use

Put CreamAPI files in the same folder as cream_setup.ps1 then just run the script

```
> ./cream_setup.ps1
Game installation path: path/to/game_folder
```

to find the id of a game you can look at the page link it's structured like this:

`https://store.steampowered.com/app/{APPID}/{NAME}`
